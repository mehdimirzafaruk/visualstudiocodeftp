# Visual Studio Code FTP Plugin

Bu proje, Visual Studio Code için açık kaynaklı bir FTP bağlantı eklentisidir. Kullanıcıların uzak FTP sunucularına kolayca bağlanmasını, dosya transferi yapmasını ve dosyalarını düzenlemesini sağlar.

## Özellikler
- Kolay FTP bağlantısı kurma
- Dosya ve klasörleri uzaktan listeleme
- Dosya indirme ve yükleme
- Dosya düzenleme ve kaydetme
- Açık kaynak kodlu ve geliştirilebilir yapı

## Kurulum
1. Bu repoyu klonlayın veya indirin.
2. Visual Studio Code'da eklenti olarak yükleyin veya geliştirme ortamınızı kurun.

## Kullanım
1. Eklentiyi etkinleştirin.
2. FTP sunucu bilgilerinizi girin.
3. Bağlantı kurulduktan sonra dosyalarınızı yönetmeye başlayabilirsiniz.
